# 001 Playback Speed Calculator

项目中文名：播放速度计算器

当前阶段：立项 / 需求定义

域名：`playspeedcalc.net`

核心关键词：

- `playback speed calculator`（播放速度计算器）

细分关键词：

- `audiobook speed calculator`（有声书播放速度计算器）
- `podcast time calculator`（播客时长计算器）
- `audiobook time calculator`（有声书时长计算器）

一句话产品定义：

帮助听有声书、播客、课程和视频的人，输入原始时长和播放倍速后，立刻知道实际耗时、节省时间和预计完成时间。

## 项目目标

第一个站主要目标是跑通闭环，不以短期赚钱为硬目标。

- L0：域名可访问，统计和收录入口配置完成。
- L1：`GSC`（谷歌搜索控制台）有收录或曝光。
- L2：真实用户完成一次计算。
