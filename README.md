# 001 Playback Speed Calculator

项目中文名：播放速度计算器

线上地址：`https://www.playspeedcalc.net`

## 项目定位

帮助听有声书、播客、课程和视频的人，输入原始时长和播放倍速后，立即知道：

- 实际耗时。
- 节省时间。
- 预计完成时间。

## 核心关键词

- `playback speed calculator`（播放速度计算器）

相关长尾：

- `audiobook speed calculator`
- `podcast time calculator`
- `audiobook time calculator`

## 项目目标

第一站主要目标是跑通完整闭环，不以短期赚钱为硬目标。

- L0：域名可访问，统计和收录入口配置完成。
- L1：GSC 有收录或曝光。
- L2：真实用户完成一次计算。
