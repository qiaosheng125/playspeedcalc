# Domain Candidates（域名候选）

项目：`Playback Speed Calculator`（播放速度计算器）

已购买域名：`playspeedcalc.net`

原则：

- 第一个站是练流程，不买高价溢价域名。
- 优先 `.com`（通用商业域名），如果价格贵或不可注册，再看 `.net`、`.app`、`.tools`、`.site`。
- 不追求完全匹配关键词，只要用户能看懂用途。
- 避免太长、难拼、带连字符太多的域名。

## 不优先

- 太长的完全匹配域名：`playbackspeedcalculator.com`（播放速度计算器完整拼写）如果很贵或太长，不强求。
- 带多个连字符的域名，例如 `playback-speed-calculator.com`，除非非常便宜且没有更好选择。
- `.xyz`（便宜通用后缀）可以作为极低成本练手，但正式感弱，不作为首选。

## 当前推荐顺序

先查这 5 个：

1. `playspeedcalc.com`（播放倍速计算器）
2. `playbackspeedcalc.com`（播放速度计算器）
3. `mediaspeedcalc.com`（媒体速度计算器）
4. `playbacktimecalc.com`（播放时长计算器）
5. `playspeed.tools`（播放倍速工具）

## 最终选择

已选择并购买：`playspeedcalc.net`（播放倍速计算器）

选择原因：

- `.net`（网络类通用域名）比 `.top`、`.xyz`、`.site` 等低价后缀更稳。
- `playspeedcalc` 短于 `playbackspeedcalculator`，容易拼写。
- 和主关键词 `playback speed calculator`（播放速度计算器）语义接近。
- 适合第一个工具站，不需要买高价 `.com`。

选择标准：

- 价格低。
- 没有溢价。
- 拼写不容易错。
- 和 `playback speed calculator`（播放速度计算器）主词有关。
- 后续能扩展到有声书、播客、课程、视频。
