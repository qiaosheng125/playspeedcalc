# SEO Plan（搜索优化计划）

## 主关键词

`playback speed calculator`（播放速度计算器）

搜索意图：用户想输入原始媒体时长和播放倍速，立刻得到实际播放时间和节省时间。

## Title / Description 草案

首页 title：

`Playback Speed Calculator - Calculate Time Saved at Any Speed`

首页 description：

`Calculate how long an audiobook, podcast, course, or video will take at 1.25x, 1.5x, 2x, or any playback speed. See time saved and finish time instantly.`

H1：

`Playback Speed Calculator`

## 页面正文必须回答

- 什么是播放速度计算器。
- 适合哪些场景：有声书、播客、课程、视频。
- 如何计算播放后的实际时长。
- 1.25x、1.5x、2x 分别适合什么场景。
- 为什么结果在本地计算，不上传数据。

## FAQ（常见问题）

- What is a playback speed calculator?
- How do I calculate time saved at 1.5x speed?
- How long does a 10-hour audiobook take at 1.25x?
- Is 2x speed good for podcasts?
- Does this calculator work for YouTube videos?

说明：FAQ 用于帮助用户理解，不把 `FAQPage`（常见问题结构化数据）当作排名策略。

## 技术 SEO 检查

- 每页唯一 title。
- 每页唯一 description。
- 每页一个 H1。
- 设置 canonical。
- 生成 sitemap.xml。
- 设置 robots.txt。
- 核心内容 SSR/SSG 或静态可索引。
- 移动端可用。
- 页面加载快。
- 隐私、条款、联系页可访问。
